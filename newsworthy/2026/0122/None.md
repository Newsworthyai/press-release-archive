# RChilli Announces Availability of Talent Data Refresh Agent on Oracle Fusion Applications AI Agent Marketplace

San Ramon, CA (Newsworthy.ai) Thursday Jan 22, 2026 @ 1:35 PM Central — RChilli, a leading provider of AI-driven solutions for recruitment transformation and Oracle partner, today announced the availability of its Talent Data Refresh Agent on the Oracle Fusion Applications AI Agent Marketplace. Seamlessly integrated with Oracle AI Agent Studio for Fusion Applications, the Oracle AI Agent Marketplace helps joint customers accelerate AI adoption at scale, boost productivity, and address complex business challenges with secure AI agents built and ready for enterprise use. With the AI Agent Marketplace, organizations can install and manage RChilli’s Talent Data Refresh Agent alongside Oracle pre-built agents in a unified experience.

 The Talent Data Refresh Agent helps organizations enhance candidate data quality and accelerate hiring decisions with the following benefits:

 * Automated Talent Enrichment: Replaces outdated profile information with structured, accurate, and up-to-date candidate data using publicly available professional sources.
* Decision-Ready Candidate Insights: Standardizes and enhances resume data to improve job matching and hiring quality.
* Integration with Oracle Fusion Cloud Human Capital Management (HCM): Fully embedded into Oracle Cloud HCM—no third-party setup needed.
* Faster, Cost-Effective Hiring: Cuts down manual effort and administrative overhead in candidate data management.

 As hiring becomes more competitive, AI is helping organizations address challenges caused by outdated or incomplete candidate data,” said Sneh Lata, Director of Products, RChilli. “With the Oracle AI Agent Marketplace, we are delivering AI-powered solutions that help our customers maintain accurate data and make more informed hiring decisions. The Talent Data Refresh Agent helps organizations keep talent data current and reliable, so recruiters can focus more on hiring rather than manual data upkeep.

 About RChilli RChilli delivers powerful AI capabilities to help recruiters automate manual tasks, drive smarter hiring decisions, and deliver exceptional candidate experiences. With certified integrations on Oracle Cloud Marketplace, SAP Store, and Salesforce AppExchange, RChilli supports over 1,600 clients globally in building agile and intelligent hiring ecosystems.
