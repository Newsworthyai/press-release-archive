# 20 States File Lawsuit Against Trump Administration for Releasing Medicaid Data to DHS

Rob Bonta, the Attorney General of California, revealed on Tuesday that 20 states have filed a lawsuit challenging the federal government’s release of private Medicaid data to deportation personnel in June. The shared data affects millions of people enrolled in Medicaid.
